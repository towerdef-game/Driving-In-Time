23. Scripting API Reference

This chapter provides a reference for FMOD Studio's scripting API, and assumes some basic knowledge of programming concepts and JavaScript. For more information about running scripts, and for example scripts, see the Scripting chapter.

The Javascript API available is split over the following modules:

  • Globals
  • Menu
  • Project
  • System
